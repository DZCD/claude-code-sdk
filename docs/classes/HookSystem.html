    Class HookSystem

    HookSystem — 钩子系统门面类。

    提供更简洁的统一 API 来管理钩子注册表。

    Example

    const hooks = new HookSystem()
    hooks.register('preTool', 'audit', async (name, input) => {
      console.log(`Tool ${name} called with`, input)
      return { allowed: true }
    })
